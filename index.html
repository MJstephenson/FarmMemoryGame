Instructions

To start the game

Click/press the start button

Choose A Game style

Click/press a a button to select a game style

  1. Select a card to start the game and flip a card
  2. select a 2nd card to flip it
  3. if the pictures match the cards stay flipped,
    pick another 2 cards
  4. if the pictures dont match the cards flip back over
  5. You can only flip 2 cards at once
  6. Win the game by matching 8 pairs of pictures
  7. Restart the game to try to beat your number of moves or your best time
